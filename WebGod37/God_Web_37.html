jQueryで実装するハンバーガーメニュー3種類

jQueryでハンバーガーメニューを実装する方法を3種類、例に挙げて紹介。 シンプルで簡単なので、初心者向け。

ハンバーガーメニューに必要な部品

「ボタン」と「メニュー本体」を用意する。
HTML/CSSで部品を作っておいて、動きの部分だけjQueryで実装することも多い。

今回PC表示の時に現れる

タグをスマホ画面の時に、ハンバーガーメニューをクリックすると＜nav＞タグが表示/非表示されるようにコードを書いていく。

  1. まず、
    タグにID「"global_nav"」をつける。
  2. 続いてCSSに記述。今回は大きく分けて2つのスタイル調整が必要

    3. 1,メニュー本体（nav要素）のスタイル調整

    ハンバーガーメニューの中身の見た目を作っていく。

    2,ハンバーボタンのスタイル調整

    メニューを開いているときのボタンデザインを「x」に変える。

  3. jquery本体の読み込みを記述。

    4. パターン1. 上からスライド $("#drawer_toggle").click(function(){ // $(this).toggleClass("open"); // $("#global_nav").slideToggle(); // });

    パターン2. フェードイン // $("#drawer_toggle").click(function(){ // $(this).toggleClass("open"); // $("#global_nav").fadeToggle(); // });

    パターン3. 横からスライド $("#drawer_toggle").click(function(){ $(this).toggleClass("open"); $("#global_nav").toggleClass("sp_open"); }); });
